# 間取りクイック3D

ブラウザ上で間取りを描き、そのまま3D表示できるシンプルな間取り作成ツールです。

## 公開URL

**公開サイト:** https://tghcgu.github.io/madori/

インストールやログインは不要です。PCのブラウザからすぐに使えます。

## 主な機能

- 部屋・壁・ドア・窓を2D画面で作成
- 円・円弧・多角形を壁として作成
- 家具・設備・階段などを配置
- `同時`・`2D`・`3D`の表示切り替え
- 2Dの間取りを3Dへ自動変換
- 複数階、屋根、影、光の向きに対応
- 部屋名の移動、寸法・色・角度の編集
- Undo / Redo
- JSONファイルへの書き出し・読み込み
- ブラウザ内への自動保存

## 基本操作

1. 左側のツールから、部屋・壁・ドア・窓・家具などを選びます。
2. 2D画面をドラッグして配置します。
3. `選択`ツールで要素を選び、移動・サイズ変更・数値編集を行います。
4. 画面上部の`同時`・`2D`・`3D`で表示を切り替えます。

### マウス・キーボード

| 操作 | 内容 |
| --- | --- |
| マウスホイール | 2D画面の拡大・縮小 |
| 右ドラッグ | 2D画面の移動 |
| `R` | 選択中の要素を90度回転 |
| `Shift + R` | 選択中の要素を15度回転 |
| `F` | 家具の左右反転・ドアの開き反転 |
| `Delete` | 選択中の要素を削除 |
| `Ctrl + Z` | 元に戻す |
| `Ctrl + Y` | やり直す |

## データの保存

作成した間取りはブラウザ内に自動保存され、外部サーバーには送信されません。

ブラウザのサイトデータを削除すると間取りも消えるため、大切なデータは上部の`書き出し`からJSONファイルとして保存してください。保存したファイルは`読み込み`から復元できます。

## ローカルで動かす

必要なもの:

- Git
- Node.js 20以上（22推奨）

```bash
git clone https://github.com/tghcgu/madori.git
cd madori
npm install
npm run dev
```

起動後、ターミナルに表示された `http://127.0.0.1:5173/` をブラウザで開きます。

Windows PowerShellで `npm` が実行できない場合は、代わりに次を使ってください。

```powershell
npm.cmd install
npm.cmd run dev
```

## ビルド

```bash
npm run build
npm run preview
```

`main`ブランチへpushすると、GitHub ActionsによってGitHub Pagesへ自動公開されます。
